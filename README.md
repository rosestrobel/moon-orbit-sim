# Moon Orbit Sim

This repo creates a simple application that simulates the moon's orbit around earth. If the user clicks on the screen, more moons will spawn.

## Overview

This project is quite simple. I wanted to start slow and work my way up towards larger simulations--eventually in C++. Python's libraries, specifically pygame, make creating GUIs much more simple compared to C++.

### Executing program

* Create a venv file to install dependencies
```
python -m venv .venv
```
* After dependencies are installed, run the program
```
python main.py
```
